# nyc-sat-analysis
Analysis of NYC public school SAT performance using Python and pandas for data manipulation and aggregation.
Project imported from DataCamp. Part of completed self-paced coursework.

# Overview:
This project analyzes New York City public school SAT performance data to uncover trends in academic achievement across schools and boroughs.
The SAT consists of three sections: Reading, Math, and Writing, each scored out of 800 (2400 total).

# Objectives:
Identify schools with the strongest math performance
Determine the top 10 schools based on total SAT score
Find which borough has the largest variation in total SAT scores

# Tools Used:
Python, Pandas 

# Key Skills Demonstrated:
Data cleaning and preprocessing
Aggregation and ranking
Basic statistical analysis (mean & standard deviation)
