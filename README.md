# nyc-sat-analysis
Analysis of NYC public school SAT performance using Python and pandas for data manipulation and aggregation. <br>
Project imported from DataCamp, and part of completed self-paced coursework. <br>

# Overview:
This project analyzes New York City public school SAT performance data to uncover trends in academic achievement across schools and boroughs. <br>
The SAT consists of three sections: Reading, Math, and Writing, each scored out of 800 (2400 total). <br>

# Objectives:
Identify schools with the strongest math performance <br>
Determine the top 10 schools based on total SAT score <br>
Find which borough has the largest variation in total SAT scores <br>

# Tools Used:
Python, Pandas 

# Key Skills Demonstrated:
Data cleaning and preprocessing <br>
Aggregation and ranking <br>
Basic statistical analysis (mean & standard deviation) <br>
